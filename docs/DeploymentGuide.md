# Deployment and User Guide

## Deployment
The project can be deployed on the following environments:

### Localhost
- Install dependencies using npm install
- Run local blockchain using Hardhat
- Deploy smart contracts locally

### Ethereum Test Network
- Connect MetaMask to Sepolia test network
- Deploy contracts using Hardhat scripts
- Use free test ETH only

## Getting Test ETH
Test ETH can be obtained from public Sepolia faucets.
These tokens are used only for testing purposes and have no real-world value.

## User Guide
Steps to use the application:
1. Install MetaMask browser extension
2. Connect wallet to the application
3. Select Ethereum test network
4. Create or join a game
5. Play the game by interacting with the UI

## Disclaimer
This application is intended for educational use only.
Real cryptocurrency or Ethereum mainnet is not used.
